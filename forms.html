Forms and Examples

Forms are written in HTML, styled with CSS and/or Bootstrap and given functionality through JavaScript and/or jQuery.

HTML

  • Use <form> elements to define an HTML form.
  • Use <input> elements to define text and radio input and to define a submit button
  • <input type="text"> will define a one-line text input field
  • <input type="radio"> will define radio buttons to let a user select one of a limited number of choices
  • <input type="submit"> will define a button for submitting a form to a form-handler.
