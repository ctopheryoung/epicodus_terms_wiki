JavaScript Arrays

JavaScript arrays are used to store multiple values in a single variable. An array is a special variable, which can hold more than one value at a time.

Example:

var numbers = [ 1, 2, 3, 4]

Adding and removing elements from Arrays

To add an element to an existing array, use the .push method. This adds the element to the end of the array.

var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.push("Lemon");
fruits;
["Banana", "Orange", "Apple", "Mango, "Lemon"]

To remove an element from an array use the .pop() method. This will remove the last element from the array (the method does not accept an argument).

var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.pop();
fruits;
["Banana", "Orange", "Apple"]
